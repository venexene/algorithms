# Min Stack

## Решение

### Два стека — O(1) на операцию, O(n)
Основной стек + minStack. Push: если значение ≤ вершина minStack (или стек пуст) — кладём и в minStack. Pop: если удаляемое == вершина minStack — pop и из minStack. GetMin: вершина minStack.

## Почему это работает
minStack хранит историю минимумов в порядке их появления. Благодаря LIFO удаление вершины восстанавливает предыдущий минимум — он уже лежит следующей в minStack. `<` не хватило бы: два одинаковых минимума подряд → первый pop удалит минимум из minStack, хотя второй элемент с тем же значением всё ещё в стеке.
