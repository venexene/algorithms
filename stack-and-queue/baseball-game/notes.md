# Baseball Game

## Решение

### Стек интов — O(n), O(n)
Стек хранит очки. `+` — pop + peek, push обратно pop + push суммы. `D` — peek × 2, push. `C` — pop. В конце суммируем стек.

## Что важно запомнить
- Стек должен хранить `int`, не `string`. Иначе `strconv` на каждой операции.
- Суммировать в конце, а не поддерживать параллельно — меньше кода, меньше шансов на ошибку.
- `Pop` + `Peek` + `Push` для `+` — правильный паттерн для «взять два последних не удаляя первый».

## Что пошло не так
1. Хранил `string` в стеке — `strconv.Itoa`/`Atoi` на каждом шаге
2. Поддерживал `sum` отдельно — двойная бухгалтерия, лишний код
