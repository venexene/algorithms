# Daily Temperatures

## Решение

### Monotonic decreasing stack — O(n), O(n)
Стек хранит индексы дней с убывающей температурой. Для каждого дня: пока температура выше чем у вершины стека — записываем разницу индексов в результат, pop. Затем push текущий индекс.

## Что важно запомнить
- Monotonic stack — стек где элементы упорядочены (здесь по убыванию температуры). Используется в задачах «следующий больший/меньший элемент».
- Хранить только индекс, не пару {temp, index}. Температуру брать из исходного массива по индексу — экономия памяти и копирования.
- Сохранять `Peek()` в переменную если используется дважды — избежать двойного вызова.
- На голом слайсе без самодельного `Stack` было бы ещё на пару процентов быстрее, но читаемость важнее.
