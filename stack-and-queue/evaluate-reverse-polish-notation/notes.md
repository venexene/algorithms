# Evaluate Reverse Polish Notation

## Решение

### Стек — O(n), O(n)
Число → push. Оператор → pop два значения, применить оператор, push результат. Порядок операндов важен: `val1 - val2`, не наоборот. В конце стек содержит единственное значение — результат.

## Что важно запомнить
- Порядок: первым идёт левый операнд (`val1`), вторым — правый (`val2`). `val1 / val2`, не `val2 / val1`.
- Классическая задача на стек — стандартный паттерн для постфиксной нотации.
