# Valid Parentheses

## Решение

### Стек — O(n), O(n)
Открывающая скобка → кладём на стек соответствующую закрывающую. Закрывающая → проверяем совпадение с вершиной стека, pop. В конце стек должен быть пуст.

## Что важно запомнить
- `map[rune]rune` для быстрого поиска пары — O(1) вместо цикла
- На стек класть закрывающую скобку, не открывающую — упрощает сравнение (просто `r == stack[top]`)
- `return len(stack) == 0` — чище чем `if/else`
- Классическая ошибка: забыть проверку стека в конце. `"["` → открывающая положилась, стек не пуст, а возвращал `true`
