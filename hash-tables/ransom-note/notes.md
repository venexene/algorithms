# Ransom Note

## Решение

### Частотный словарь — O(m + r), O(1)
Мапа: символ → количество в magazine. Проходим по ransom — вычитаем. Если счётчик ушёл в минус — букв не хватило.

### Массив частот — O(m + r), O(1)
`[26]int` вместо мапы. Символ − `'a'` = индекс. Быстрее мапы (прямой доступ, нет хеширования). Работает только для ASCII-строчных букв.

## Что важно запомнить
- Тот же паттерн что Valid Anagram: частоты + вычитание. Только здесь строки разной длины и не требуется полное совпадение.
- Ранний выход: `if count < 0 → return false` — не ждём конца цикла.
- `r - 'a'` (одинарные кавычки), не `r - "a"` (двойные). Руна минус руна = число. Руна минус строка = ошибка компиляции.
