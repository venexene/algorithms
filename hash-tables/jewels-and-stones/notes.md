# Jewels and Stones

## Решение

### Хеш-сет — O(J + S), O(J)
Драгоценности → сет. Камни → счётчик совпадений.

## Что важно запомнить
- Простейший паттерн «сет для быстрого lookup». Основа для более сложных задач.
- `for _, r := range "abc"` даёт `rune`, не `byte`. Для ASCII-строк разницы нет, для Unicode — критично.
