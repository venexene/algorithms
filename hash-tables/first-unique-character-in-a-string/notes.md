# First Unique Character in a String

## Решение

### Два прохода с частотами — O(n), O(1)
Первый проход: считаем частоты символов. Второй проход: ищем первый символ с частотой 1. Два прохода по строке (не по мапе — мапа не сохраняет порядок).

## Что важно запомнить
- **Искать первый уникальный надо по строке, не по мапе.** Итерация по мапе — случайный порядок.
- Сет не работает: символ с частотой 3+ будет то добавлен, то удалён, в итоге останется в сете как «уникальный». Нужен счётчик, не сет.
- Массив в Go — value type. `val := mp[k]` создаёт копию, изменения теряются. Если нужно менять значение внутри мапы — либо `mp[k] = newVal`, либо храни указатель/слайс.
