# Merge Sorted Array

## Решение

### Три указателя с конца — O(m + n), O(1)
`n1` — конец значимых элементов в nums1. `n2` — конец nums2. `i` — позиция записи с хвоста nums1. Сравниваем наибольшие элементы nums1 и nums2, ставим больший в `nums1[i]`, сдвигаем соответствующий указатель.

Границы: `n2 < 0` → всё из nums2 на месте, выходим. `n1 >= 0` → nums1 ещё не кончился, сравниваем. Иначе берём из nums2.

## Что важно запомнить
- **Идти с конца** — неочевидный трюк. Свободное место в хвосте nums1 подсказывает направление.
- Если бы шли с начала — пришлось бы сдвигать элементы nums1, что дало бы O(n²).
- `n2 < 0` — достаточно для выхода. Если `n1 < 0` — просто докидываем остаток nums2, это обрабатывается через `else`.
