# Two Sum II — Input Array Is Sorted

## Решение

### Два указателя — O(n), O(1)
Массив уже отсортирован. Левый указатель с начала, правый с конца. Сумма меньше target → левый вправо. Сумма больше → правый влево. Нашли — вернули индексы + 1 (1-based).

## Что важно запомнить
- В отличие от обычного Two Sum, здесь два указателя — оптимальное решение. Сортировка не нужна, O(n) и O(1) памяти. Хеш-таблица дала бы O(n) и O(n) — хуже.
- `if / else if / else` — именно такая цепочка, не три отдельных `if`. Как в Valid Palindrome.
- Индексы в ответе 1-based, не 0-based. Забыть +1 — частая ошибка.
