# Minimum Size Subarray Sum

## Решение

### Sliding window — O(n), O(1)
Окно расширяется вправо (`right++`). Как только сумма ≥ target — сужаем слева до тех пор пока сумма ≥ target, на каждом шаге записывая длину. Возвращаем минимальную длину.

## Что важно запомнить
- Переменное окно: right расширяет, left сужает. Не фиксированный размер как в Maximum Average.
- `res = len(nums) + 1` как значение превышающее возможное для «не найдено». В конце проверка → 0.
- `for sum >= target` — естественная остановка сужения. Не надо считать шаги.
- `res` обновляется ДО вычитания — пока окно ещё валидно.

## Что пошло не так
1. Схлопывал окно в точку: `left = right; sum = 0` — терял потенциально более короткие подмассивы с более ранним left
2. Внутренний цикл `for left < right` без проверки суммы — съедал все элементы, даже когда сумма уже ниже target
3. Итог: `for sum >= target` — правильный критерий остановки сужения
